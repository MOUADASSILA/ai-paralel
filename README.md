# Paralel Project

## Project Structure

- `/Paralel` - Frontend application (Svelte, Tailwind CSS, Vite)
- `/back` - Backend application (Node.js)

## Setup Instructions

### Frontend

```bash
cd Paralel
npm install
npm run dev
```

### Backend

```bash
cd back
npm install
npm start
```

## Technologies Used

- Frontend: Svelte, Tailwind CSS, Vite
- Backend: Node.js, Express
